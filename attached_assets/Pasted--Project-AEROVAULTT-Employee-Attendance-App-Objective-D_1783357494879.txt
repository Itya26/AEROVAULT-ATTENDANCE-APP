# Project: AEROVAULTT Employee Attendance App

## Objective

Develop a modern, production-ready employee attendance application for **AEROVAULTT Intelligence Private Limited**.

The application should allow employees to mark their own attendance only when they are physically present at the office using GPS-based geofencing.

The UI should be clean, minimal, responsive, and suitable for both Android and desktop browsers.

# GPS Location Verification

When the employee opens the attendance page:

* Request location permission.
* Obtain the current GPS coordinates.
* Compare the employee's location with the office location.

Office Location should be configurable from the Admin Panel.

Example:

Latitude:
(Admin configurable)

Longitude:
(Admin configurable)

Allowed Radius:
100 meters (configurable)

If employee is inside the allowed radius:

Display

Office Location Verified

Enable the Check In button.

If outside:

Display

You are outside the office premises.

Disable Check In.

Store:

* Latitude
* Longitude
* GPS Accuracy
* Timestamp

# Attendance Workflow

## Check In

Employee presses Check In.

Store:

* Employee ID
* Date
* Check In Time
* GPS Coordinates
* GPS Accuracy
* Device Information
* Attendance Status

Disable Check In after successful attendance.

---

## Check Out

Employee presses Check Out.

Store:

* Check Out Time
* Total Working Hours

Attendance becomes completed.

---

# Attendance Rules

One Check In per day.

One Check Out per day.

Prevent duplicate attendance.

If employee forgets Check Out:

HR can manually close attendance.

# Security

Prevent attendance without GPS permission.

Prevent attendance if GPS accuracy is poor (greater than configurable threshold).

Detect mock/fake GPS where supported and reject attendance if detected.

Use HTTPS.

Encrypt passwords.

JWT authentication.

Audit logs for all attendance modifications.
